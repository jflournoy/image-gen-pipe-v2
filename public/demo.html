üîç What is Beam Search?

Beam Search is an iterative refinement algorithm that explores multiple candidate solutions in parallel:

  • Iteration 0: Generate N initial candidates exploring different creative directions
  • Iterations 1+: Keep the best M candidates and refine each one further
  • Ranking: Uses pairwise image comparison to find the highest-quality final result
  • Result: Returns the winning candidate that evolved through multiple iterations

üí° Think of it like a tournament: Start with many competitors, eliminate weaker ones each round, and refine the best ones.
